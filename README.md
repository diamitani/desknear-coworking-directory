# DeskNear - Coworking Space Directory

DeskNear is a lightweight, frontend-only directory application for discovering coworking spaces, private offices, and meeting rooms.

## Tech Stack
- **HTML5**
- **Tailwind CSS** (via CDN) for rapid, responsive styling
- **Alpine.js** (via CDN) for reactive state management (search, filtering, data binding)
- **FontAwesome** for icons

## Features
- **Responsive UI:** Works on mobile, tablet, and desktop.
- **Instant Search:** Filter spaces by name or neighborhood dynamically.
- **Advanced Filtering:** Filter spaces by amenities (24/7 Access, Meeting Rooms, etc.) and price ratings ($, $$, $$$).
- **Mock Data:** Pre-populated with beautiful sample data ready for a real backend integration.

## How to Run
Simply open `index.html` in your favorite web browser. No build steps required.

## Deployment
This project includes a `vercel.json` and is ready to be deployed as a static site on [Vercel](https://vercel.com).
